# Быстрый запуск веб-приложения

## 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

## 2. Запуск сервера
```bash
python -m uvicorn api:app --reload --host 127.0.0.1 --port 8000
```

## 3. Открытие веб-приложения
Перейдите в браузере по адресу: http://127.0.0.1:8000

## Возможности:
- Выбор города из списка или поиск
- Прогноз численности населения на 5-15 лет
- Аналитическая справка с AI-генерацией
- Интерактивные графики и метрики качества
