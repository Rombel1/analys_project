# Демографическая аналитика Российской Федерации

## Веб-приложение для анализа и прогнозирования населения

### 🚀 Быстрый запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите сервер:**
   ```bash
   python -m uvicorn api:app --reload --host 127.0.0.1 --port 8000
   ```

3. **Откройте веб-приложение:**
   - Перейдите по адресу: http://127.0.0.1:8000
   - Выберите город из списка или используйте поиск
   - Получите прогноз или аналитическую справку

### 📊 Возможности веб-приложения

- **Выбор города:** Поиск по названию или выбор из списка по регионам
- **Прогноз населения:** На 5, 10 или 15 лет с доверительными интервалами
- **Аналитическая справка:** Полный отчет с тенденциями, факторами и рекомендациями
- **Интерактивные графики:** Визуализация прогнозов с помощью Chart.js
- **Метрики качества:** MAPE, RMSE, MAE, R² для оценки точности модели

### 🔧 API для разработчиков

**Документация API:** http://127.0.0.1:8000/docs

#### Основные эндпоинты:

- `GET /api/regions` - список регионов
- `GET /api/cities` - список городов (с фильтрацией по региону)
- `GET /api/cities/search?q={query}` - поиск городов
- `GET /api/forecast/{city}` - прогноз для города
- `POST /api/forecast/compare` - сравнение прогнозов
- `GET /api/ai/report/{city}` - аналитическая справка (JSON)
- `GET /api/ai/report/{city}/markdown` - аналитическая справка (Markdown)

### 📁 Структура данных

Поместите CSV файлы в папку `data/`:
- `cities_historical_2015_2024.csv` - исторические данные
- `cities_forecast_2024_2039.csv` - прогнозные данные (опционально)

### 🛠 Консольное приложение

Для запуска консольной версии (как в оригинальном main.py):
```bash
python main.py
```

- Python 3.9+
